# Dev_Tech
Dev_Tech is a full-stack AI monorepo and the home of the Recursive Understanding Engine (RUE). Built with React/Vite and Node.js, it uses Google Gemini to power recursive, AI-driven concept exploration. The stack includes TurboRepo for builds, Drizzle ORM with Neon PostgreSQL for data, and supports auth, sessions, and breadcrumb navigation.
